# 🚣 Math River Crossing – Bangladesh Edition 🇧🇩

A fun and interactive math learning game inspired by children in Bangladesh.  
Players solve math problems to move the boat across the river before time runs out!

---

## 🎮 Game Features

- 🧮 Random math questions (Addition, Subtraction, Multiplication)
- 🎚️ Multiple difficulty levels (Easy, Medium, Hard)
- ⏳ 60-second countdown timer
- ⭐ Live score tracking
- 🏆 High score saved using Local Storage
- 🚣 Boat movement animation based on answers
- 🎨 Beautiful Bangladesh-themed gradient design

---

## 🛠️ Technologies Used

- HTML5  
- CSS3  
- JavaScript (Vanilla JS)  
- LocalStorage API  

---

## 🚀 How to Run the Project

1. Download or clone the repository
2. Make sure all files are in the same folder:
   - `index.html`
   - `style.css`
   - `script.js`
3. Open `index.html` in your browser
4. Start playing 🎉

## 💡 Game Objective

Solve math problems correctly to move the boat forward.  
Wrong answers move the boat backward.  
Reach the end before time runs out to win!
